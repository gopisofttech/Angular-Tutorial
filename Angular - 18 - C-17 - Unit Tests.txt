----- Unit Tests -----


-> Unit Tests

	-> Reduce regression errors at the object or function level
	-> Clarify the intent of your code
	-> Usually does NOT test the user interface or user interactions

-> End to End(e2e) Tests

	-> Also called integration tests
	-> Tests at the component and page level
	-> Focuses on the expectations of your end users

-> Testing Guidelines

	-> Always test your code before sharing it
	-> Know your target code coverage

-> Recommended Setup

	-> Unit Testing

		-> Jasmine testing framework
		-> Karma test runner

	-> End to End testing

		-> Protractor

  {

	const pipe = new HightlightTextPipe();

	}
	);

	const pipe = new HightlightTextPipe();
